Roadmap

Where this project is going next.

SHIPPED (v1.0)

SIR compartmental simulator with dark-themed dynamic charting.
Geographic spread visualization on Leaflet with wave rings and secondary clusters.
NewsAPI integration with graceful synthetic fallback.
Pre-configured scenario library (measles, COVID-19, flu, Ebola, pertussis, common cold).
Sensitivity analysis sweeps over β or γ.
Vaccination what-if comparison via initial-recovered fraction.
Browser-local simulation history.

PLANNED (v1.x)

SEIR model with an Exposed compartment and configurable incubation period.
Real CDC FluView dataset integration for empirical comparison.
Parameter estimation from observed case curves via least-squares fit.
Multi-strain dynamics with cross-immunity matrix.
Mobile-responsive map and chart redesign.
Word-boundary regex matching in news severity tagging.
Persistent server-side history for opt-in users.

RESEARCHING (FUTURE)

Population-density-weighted spread using raster data.
Real flight-network data for international cluster placement.
Agent-based model comparison alongside the compartmental view.
Bayesian parameter inference with pymc.
Intervention scheduling: time-varying β for lockdowns and behavioral change.
Spatially explicit metapopulation model with commuter mixing matrices.
